Project 1: Beyster 1695 Simulator (Gold Edition)

A VR preservation of the historic Beyster 1695 Laboratory. This project features a fully interactive environment with custom mechanics, NPC dialogue systems, and complex physical interactions[cite: 13, 14].

Unreal Engine 5.2 Blueprints Meta Quest VR

Technologies & Implementation

Developed using Unreal Engine 5 and Blueprints. Key technical implementations include LineTrace-based interaction systems, FInterp To for a smooth, interactive Day-Night cycle, and World-Space UI Widgets for non-diegetic NPC communication.

Core Project Tasks

  • Smooth Day-Night Time Progression
  • Physics-based Trash & Disposal System
  • 3 Unique NPCs with Cycling Dialogue
  • Interactive Whiteboard Drawing Mechanics
  • Procedural Copy Machine Item Duplication
  • Environmental Toggles (Fire Alarms/Lights)

Custom Gold Interaction: "The Lab Technician"

For the Gold deliverable, I implemented a multi-stage maintenance workflow. Players act as lab staff to resolve environmental crises: fixing a sparking printer, resolving a glitching computer, and plugging a USB drive into a server to successfully complete the 3-task sequence and unlock the final reward.
