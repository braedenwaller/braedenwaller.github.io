Project 1: Beyster 1695 Simulator (Gold)

A high-fidelity VR reconstruction of the historic Beyster 1695 Laboratory. This project demonstrates environmental block-outs, complex interaction systems, and custom gameplay logic implemented for the EECS XR Capstone[cite: 13, 14].

Unreal Engine 5.2 Blueprints VR Interaction

Technologies & Techniques

Utilized Unreal Engine 5 and Blueprints to develop a modular interaction system[cite: 40]. Key implementations include LineTrace-based affordances, FInterp To for smooth day-night transitions, and UMG Widget Components for world-space NPC dialogue.

Core Implemented Features

  • Smooth Day-Night Sun Progression
  • Dynamic NPC Dialogue Cycle (3 Unique NPCs)
  • Physics-based Trash & Disposal System
  • Interactive Whiteboard Drawing
  • Functional Fire Alarm & Light Toggle Systems
  • Procedural Copy Machine Duplication

Custom Gold Interaction: Lab Assistant Task

Implemented a custom "Lab Assistant" workflow. Players must resolve environmental issues—such as glitching computers and sparking printers—before delivering a Lab Report to an NPC to receive a celebratory "Green Light" reward and auditory feedback.
