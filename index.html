Project 1: Beyster 1695 Simulator (Gold Edition)

A VR preservation of the historic Beyster 1695 Laboratory. This project features a fully interactive environment with custom mechanics, NPC dialogue systems, and complex physical interactions.

Unreal Engine 5.2 Blueprints Meta Quest VR

Technologies & Implementation

Developed using Unreal Engine 5 and Blueprints. Key technical implementations include LineTrace-based interaction systems, FInterp To for a smooth, interactive Day-Night cycle, and World-Space UI Widgets for non-diegetic NPC communication.

Core Project Tasks

  • Smooth Day-Night Time Progression
  • Physics-based Trash & Disposal System
  • 3 Unique NPCs with Cycling Dialogue
  • Interactive Whiteboard Drawing Mechanics
  • Procedural Copy Machine Item Duplication
  • Environmental Toggles (Fire Alarms/Lights)

Custom Gold Interaction: "The Lab Technician"

For the Gold deliverable, we implemented a multi-stage maintenance workflow. Players act as lab staff to resolve environmental crises: fixing a sparking printer, resolving a glitching computer, and plugging a USB drive into a server to successfully complete the 3-task sequence and unlock the final reward.

Project 2: A2-GO! (Gold Edition)

A location-based augmented reality mobile application inspired by Pokémon Go, built with Unity Engine, Mapbox GPS integration, and ARFoundation. Players explore a real-world map of Ann Arbor, discover landmarks, plant and grow virtual trees, manage an in-game economy, and defend their trees from squirrels in AR combat.

Unity 2021 C# Scripting ARFoundation / ARKit Mapbox SDK iOS

Technologies & Implementation

Developed using Unity 2021 with C# scripting. The app integrates Mapbox SDK for real-world GPS map rendering and ARFoundation with ARKit for camera-based AR interactions. Key technical implementations include ARRaycastManager for plane detection and cursor placement, GPS-to-WorldPosition coordinate conversion via Mapbox's AbstractMap API, a static TreeBank data system for cross-scene tree persistence, and DontDestroyOnLoad for persistent UI across scene transitions.

Core Project Tasks

  • AR Seed Placement via Cursor Raycasting
  • Scrollable Inventory Bar with Quantity Tracking
  • 6 Unique Tree Types with Gradual Growth
  • GPS Coordinate Conversion (AR to Map)
  • Landmark Discovery & Seed Unlocking
  • Seed Shop with Currency Economy
  • Squirrel Spawning & Evasion AI
  • Acorn Projectile Combat System
  • Dynamic Music Switching (Explore / Interact / Battle)
  • Orbit Camera with Touch Controls
  • Compass & Currency UI Overlays
  • Image Tracking Currency Boost

Custom Gold Feature: Crabapple Bonus Drop

For the Gold deliverable, we implemented a custom mechanic where fully-grown Crabapple trees drop a tappable red apple that grants bonus currency when collected. The TreeLogic script monitors the tree's growth scale and spawns the apple prefab once it reaches maturity, while an AppleClick script uses Physics.Raycast to detect mobile screen taps on the 3D object in AR space.

My Contributions

I was responsible for the AR interaction pipeline: seed placement logic, inventory bar UI, the SeedManager and TreeBankManager scripts for cross-scene tree persistence, audio systems for both exploration and interaction modes, acorn projectile combat, the SquirrelManager random spawning system, and the Crabapple bonus drop mechanic. My partner handled landmarks, the walking avatar, orbit camera controls, the seed shop, and 3D tree modeling.

Project 3: Space Runner VR

A sci-fi themed VR workout game built in Unreal Engine where players physically dodge obstacles in real time. As a team contribution, I implemented the wall cutout room mechanic — walls with body-shaped gaps that slide toward the player, forcing them to physically move their body to pass through safely.

Unreal Engine Blueprints Meta Quest Pro VR Fitness

Technologies & Implementation

Built in Unreal Engine using Blueprint scripting for the VR template. The wall cutout mechanic uses a Timeline + Lerp approach for smooth, predictable wall movement toward the player, with OverlapAllDynamic collision detection and a Cast To VRPawn check to reliably identify player hits. A bHasHitPlayer boolean guards the hit logic to prevent double-triggering on a single pass.

My Contributions

  • Wall Cutout Room Mechanic (Blueprint)
  • Timeline-Driven Wall Movement
  • VRPawn Collision & Hit Detection
  • Double-Hit Prevention Logic
  • VR Spawn Configuration (PlayerStart)
  • Room Integration into Team Build

Team & Role

Space Runner VR is a group project with teammates handling 3D modeling and core mechanics, level assembly and the devlog website, and the health/lives system. I joined the team mid-stream and contributed room implementation, rebuilding the wall cutout mechanic from scratch in the shared project after a hardware change. The team collaborates via Discord and a shared repo.
